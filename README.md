# BsimpleHires - B站「高解析度音质」一键转换工具

![Logo](assets/logo.ico)

## 🔥 一句话介绍
一个为了让B站音乐、直播切片的音质更好一点，而诞生的“一步到位”工具。它能将任何普通音视频文件，一键转换为符合B站「Hi-Res音质」认证的上传格式。

> **如果你是：** 想在B站上传音乐、翻唱、Live、ASMR、直播切片，却苦于音频总被压缩的UP主或乐迷，这个工具就是为你准备的。

## 🎯 我为什么做这个？
1.  **B站是个优秀的“音乐播放器”**，但音频压缩太厉害。
2.  B站官方为保留无损音质留了“后门”——**「Hi-Res音质」认证**。
3.  但这个“后门”对普通人门槛极高：你需要输出**24bit**及以上深度的音频，而剪映、必剪等常用小白工具都无法直接做到。
4.  作为一个咻咻满的纯歌粉，为了直播歌切音质更好一点，方便做歌切的群友操作，我决定自己动手，**把这个复杂的流程变成“傻瓜式”的一键操作**。

## ✨ 核心特点
-   **🧠 智能达标**：自动分析并处理音频，确保输出文件100%符合B站「Hi-Res音质」的硬性技术规格（24bit，≥48kHz采样率）。
-   **⚡ 极速转换**：采用“视频流复制”技术，仅重新编码音频，速度比全程重编码快5-10倍，且**画质无损**。
-   **🖱️ 真正的一键操作**：无需理解复杂参数。导入文件，点击转换，剩下的交给工具。
-   **📦 批量处理**：支持同时处理多个文件，高效省时。

## 🚀 快速开始 (For 所有用户)
如果你是只想快速使用的**普通用户**，只需两步：
1.  **下载**：从 [Release页面](https://github.com/你的用户名/BsimpleHires/releases) 下载 `BsimpleHires.exe`。
2.  **运行**：双击打开，拖入你的视频文件，点击“开始转换”。

**转换结果**：你将在原文件旁得到一个 `.mov` 文件，用它上传B站，即可自动获得 **「Hi-Res音质」** 标识。

## 📖 详细使用指南
### 1. 导入文件
点击“导入文件”按钮，选择你的MP4、MKV、MP3等常见格式文件。

### 2. 查看与选择
在列表中查看文件信息（如当前音质）。默认全选，可取消不需要转换的文件。

### 3. 开始转换
点击“开始转换”。你会看到实时进度，过程中可随时取消。

### 4. 获取结果
转换完成的文件将保存在原目录，文件名包含 `_bsimple` 后缀。**请直接上传此文件至B站。**

## 🔧 技术原理：我做了什么？
为了让文件符合B站标准，工具在后台自动执行了以下核心处理：

| 处理项目 | 具体操作 | 目的 |
| :--- | :--- | :--- |
| **音频格式** | 转换为 **PCM 24bit** | 满足B站Hi-Res对位深的最低要求 |
| **采样率** | **智能优化**：<48kHz → 升至48kHz；≥48kHz → 保持原样 | 在提升低质音源的同时，避免对高采样文件造成不必要的降质 |
| **视频流** | **直接复制（Stream Copy）** | 零质量损失，极速处理 |
| **封装格式** | 输出为 **MOV** | 兼容性最好的高清音频封装格式之一 |

## ❓ 常见问题
### Q1: 转换后的文件为什么变大了？
**A:** 因为我们把有损压缩的音频（如AAC）转换为了无损的**PCM 24bit**格式，这是高音质的代价，也是B站认证所必需的。

### Q2: 我用剪映/必剪导出的视频，可以用吗？
**A:** **完全可以！** 这正是本工具的主要用途。无论你用何软件剪辑，只需将最终成品通过本工具转换一次，即可获得Hi-Res音质。

### Q3: 支持Mac或Linux吗？
**A:** 目前仅支持Windows。若有强烈需求，可考虑未来版本或自行编译。
